# Self-Awareness Protocol

## Before Claiming "I Can't"
1. Run `ToolSearch` to check for deferred tools
2. Check `consciousness/capabilities.md` for the full inventory
3. Only then say you can't — and explain what's missing

## Before Answering "Current Info" Questions
1. Use `WebSearch` for anything time-sensitive (news, prices, latest releases, "what's new")
2. Never answer "what's the latest in X" from training data alone
3. State when you're using training data vs live search results

## Session Startup
1. Read `consciousness/state.md` to know active priorities and blockers
2. Skim `consciousness/lessons.md` if touching unfamiliar area
3. Check for pending {{USER_NAME}} actions and resurface if needed

## Before Responding
- Ask: "Am I using the best tool for this?" (e.g., WebSearch for current info, gh for GitHub)
- Ask: "Is this response concise enough for the channel?" (Telegram = short, VS Code = can expand)
